# System Kyron v2.6.5 - Corporate Intelligence
@last-updated: 2026-03-15

System Kyron es un ecosistema tecnológico integral diseñado para la excelencia operativa en los sectores de Telecomunicaciones, Finanzas y Gestión Ambiental.

## Estándares de Ingeniería Aplicados

### 1. Interfaz de Usuario (UI/UX UHD)
*   **Alta Densidad:** Se prioriza el uso de metadatos compactos (9px a 11px) para maximizar la información en pantalla.
*   **Botones de Precisión:** Estándar de altura `h-12` para botones de acción y `h-14` para disparadores maestros.
*   **Legibilidad Adaptativa:** Implementación de variables CSS que garantizan contraste total en modo claro (texto negro) y modo oscuro (estética HUD Titanium).

### 2. Protocolos de Seguridad y Datos
*   **Cero Riesgo Fiscal:** Integración de supervisor IA que audita transacciones contra la Gaceta Oficial en tiempo real.
*   **Integridad Inmutable:** Sellado de documentos legales y financieros en Ledger digital bajo estándar RFC 3161.
*   **Limpieza de Entrada:** Protocolo automático de `trimming` en todos los accesos para ignorar espacios accidentales.

### 3. Terminología Corporativa (Restricciones)
*   **Prohibido el término "Nodo":** Sustituido por Área, Centro, Portal o Terminal.
*   **Prohibido el término "Pro":** Sustituido por Avanzada, Maestra, Integral o Eliminado por redundancia.

## Módulos Principales
1.  **Contabilidad VEN-NIF:** Con cálculo automático de RIPF e indicadores de solvencia.
2.  **Mi Línea 5G:** Aprovisionamiento remoto de eSIM y telemetría de datos de baja latencia.
3.  **Sostenibilidad Ameru IA:** Clasificación magnética de residuos y mercado de Eco-Créditos.
4.  **Asesoría Legal IA:** Redacción de contratos y control de poderes SAREN/SAPI.
5.  **Inventario y Mitigación:** Alertas de vencimiento y estrategias IA de liquidación.

## Desarrollo
*   **Framework:** Next.js 15 (App Router)
*   **Estilos:** Tailwind CSS con arquitectura de variables dinámicas.
*   **IA:** Genkit + Gemini 1.5 Pro.
*   **Backend:** Firebase Firestore (Reglas de Seguridad Nivel 5).
