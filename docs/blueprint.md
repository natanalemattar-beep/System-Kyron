# **App Name**: FinTrack Automation

## Core Features:

- Automated Data Entry: Automatically extract and input data from financial documents using OCR technology.
- Transaction Categorization: Use machine learning to automatically categorize transactions for accurate financial tracking.
- Report Generation: Generate financial reports (e.g., income statement, balance sheet) with customizable periods.
- Dashboard Overview: Display key financial metrics in a customizable dashboard for at-a-glance insights.
- Invoice Management: Create, send, and track invoices, automating the accounts receivable process.

## Style Guidelines:

- Primary color: Teal (#26A69A), symbolizing trust and growth, reflecting the app's core purpose.
- Background color: Light gray (#F0F4F3), providing a clean, professional backdrop that minimizes distraction.
- Accent color: Soft Green (#80CBC4) to guide the users to relevant actions, forms and processes.
- Font pairing: 'Inter' (sans-serif) for both headlines and body, to convey simplicity.
